# ConfigTrustLevel

**`Enumeration`**

The amount of trust the client should give to contacts

## Enumeration Members

### All

> **All**: "always"

Always trust new identities

### None

> **None**: "never"

Never trust new identities

### OnFirstUse

> **OnFirstUse**: "on-first-use"

Trust new identities on first use
